# bjmm.github.io

This repository hosts my personal portfolio website.

## Description

A simple HTML and CSS site showcasing my background, education, projects, and skills. The site is meant to be lightweight and contains no build steps or external dependencies.

## Installation

Clone or download this repository. Open `index.html` in your preferred browser to view the site locally.

## Dependencies

None. The project uses plain HTML and CSS without additional frameworks or libraries.
